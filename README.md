## Jupyter notebook快捷键

### 命令模式和编辑模式切换

鼠标点击任意一个```cell```或者按```Enter```回车键，进入编辑模式

鼠标点击```cell```区域外的任意一个位置或者直接按```Esc```键进入命令模式

### markdown和代码块的切换

在```命令模式```，```m```进入markdown模式、```y```则进入代码模式
